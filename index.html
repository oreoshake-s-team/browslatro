Browslatro — a Balatro-style card game in your browser

Browslatro is a free, browser-based take on the poker roguelike Balatro. Build a deck, play scoring poker hands, and beat escalating blinds using jokers, vouchers, tarot, planet, and spectral cards.

  • Poker-hand scoring with chips and multipliers
  • Jokers, vouchers, and boss blinds
  • Tarot, planet, and spectral consumables
  • Plays entirely in your browser — no install required
  • Suggested moves via machine learning
  • AI integration for explaining suggested moves or generating new moves entirely using Anthropic's API (rate-limited public use, unlimited BYOK use)
